# School

This repository contains school-related projects and assignments.

## Project Structure

- `backend/` - Backend server and API
- `frontend/` - Frontend application

## Setup

1. Clone the repository
2. Install dependencies:
   - Backend: `cd backend && npm install`
   - Frontend: `cd frontend && npm install`
3. Run the development servers

## Contributing

Please ensure all sensitive data (API keys, passwords, etc.) are stored in environment variables and not committed to the repository.
